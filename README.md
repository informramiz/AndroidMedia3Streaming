# AndroidMedia3Streaming
An app that streams video and audio using the latest Media3 libraries and showcase different use cases in action.

## Demo

![demo](docs/demo.gif)

## Concepts Covered

This app covers the following concepts.

1. Use of the new `Media3 Android libraries` to build a playlist of video + audio and stream them with the native Android ExoPlayer.
2. Show how to use `Media3` with Android `Jetpack Compose`
3. Use of Navigation Component in Compose
4. Writing `lifeycycle-aware` composables
5. `Hilt` for dependency injection in compose.
6. State in Compose
7. Use of Scafold layout of Material Design 3.
8. Material Design 3.
